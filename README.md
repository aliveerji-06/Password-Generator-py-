# Password Generator:

This is a simple password generator I built in Python to get more hands-on experience with the language while creating something useful. It runs in the terminal and lets you generate secure passwords with different customization options. You can also save generated passwords, view them later, or remove them when they're no longer needed.

### What it can do:

Generate passwords of any length, include uppercase letters, numbers, and special characters, create multiple passwords at once, check password strength, and save, view, or delete passwords from a text file.

### Built with:

Python 3 using the built-in random, string, and os modules.

### What I learned:

Building this project helped me become more comfortable working with user input, file handling, and writing cleaner, reusable Python code. It was also a good introduction to structuring a small command-line application from start to finish.
