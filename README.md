# 🔐 Password Generator

A simple command-line password generator built in Python. This project was created by me to practice working with Python's built-in libraries, user input, loops, and file handling while building something that's actually useful. It allows users to generate secure, customizable passwords and manage saved passwords directly from the terminal.

## Features

- Generate passwords of any length
- Choose whether to include:
  - Uppercase letters
  - Numbers
  - Special characters
- Generate multiple passwords in a single run
- Display a basic password strength rating (Weak, Medium, Strong, or Very Strong)
- Save generated passwords to a text file
- View or delete saved passwords without leaving the program

## Technologies Used

- Python 3
- Standard libraries:
  - `random`
  - `string`
  - `os`

## What I Learned

Building this project helped me get more comfortable with:

- Working with Python's `random` and `string` modules
- Validating user input
- Reading from and writing to files
- Organizing code into reusable functions
- Creating a simple menu-driven command-line application
